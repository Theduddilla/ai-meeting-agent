# AI Meeting Agent

## Project Overview
The AI Meeting Agent is an innovative solution designed to enhance the efficiency and effectiveness of meetings. It utilizes advanced AI technology to automate various aspects of meeting management, helping teams focus on what matters most.

## Features
- **Automated Scheduling:** Seamlessly schedule meetings based on participants' availability.
- **Meeting Summarization:** Provide concise summaries after meetings, capturing key points and action items.
- **Follow-Up Reminders:** Automated reminders for action items agreed upon during meetings.
- **Integration Capabilities:** Connect with other productivity tools to streamline workflows.

## Technology Stack
- **Programming Language:** Python
- **AI Frameworks:** TensorFlow, PyTorch
- **Database:** PostgreSQL
- **Front-end:** React
- **Hosting:** AWS

## Quick Start Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Theduddilla/ai-meeting-agent.git
   ```
2. Change into the directory:
   ```bash
   cd ai-meeting-agent
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
